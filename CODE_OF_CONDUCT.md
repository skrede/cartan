# Code of Conduct

This project has adopted the **Contributor Covenant, version 2.1** as
its code of conduct. The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
(also archived at <https://www.contributor-covenant.org/version/2/1/code_of_conduct.txt>).

## Scope

The code of conduct applies within all project spaces -- the GitHub
repository (issues, pull requests, discussions), any official
communication channels, and public events where an individual is
representing the project.

## Enforcement

Reports of unacceptable behavior may be sent to the project maintainer
at **code@skrede.me**. All reports will be reviewed and
investigated promptly and fairly. Reporters' privacy is respected.

Enforcement follows the **Community Impact Guidelines** described in
Section 5 of the canonical Contributor Covenant 2.1 text, which sets
out the progression from correction through warning, temporary ban, and
permanent ban for repeated or severe violations.
